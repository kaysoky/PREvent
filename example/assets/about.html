Bluegiga™ Smart Application on Android OS

Version: 1.0.2

This is Bluegiga's Bluetooth Smart (4.0) tool application for Bluegiga Smart (BLE) modules. The application is running on Android 4.3 and later releases. The application and source code are available at http://www.bluegiga.com/support.

More information about Bluegiga Bluetooth 4.0 products is available at www.bluegiga.com/products/.

Open source components used:

  • tree-view-list-android - This is a small widget that provides quite configurable tree view list. It is based on standard android list view. The widget is fully configurable - with your custom adapter you can provide your own implementation of the item views - even completely different implementation of item views depending on the tree level you are at. Implementation follows best approach of Adapters from android so views at the same tree level are reusable.
